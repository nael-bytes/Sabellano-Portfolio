Traffic Slight

The Traffic Slight mobile application is a comprehensive capstone project designed to create a smarter, more efficient commuting experience for motorcycle riders, specifically targeting traffic-heavy urban areas like Valenzuela City. Leveraging the Google Maps API, the app provides live traffic updates and generates optimized route suggestions that balance minimizing travel time with maximizing fuel efficiency. Key rider-focused features include detailed fuel consumption tracking via odometer or automated distance calculation, personalized motorcycle profiles, and critical maintenance alert systems that prompt users for refueling, tuning, and specific oil changes (tracked by distance based on oil type). Furthermore, it integrates current motorcycle fuel prices, which users can report changes to, and provides post-ride trip summaries detailing distance and fuel used. The entire system is managed by an administrative backend that oversees user accounts, data integrations, and fuel price accuracy, ensuring the application remains a reliable, data-driven tool for riders seeking to reduce delays and lower operating costs.
